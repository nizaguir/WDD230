White Space
& Clean Design

Zara

zara.com

This website uses the white space principle by using proper white space between elements. For example, there are columns with purposeful borders in order to distinguish a new are of focus. Overall, the white space and clean design of the website helps with readability between content.

Contrast

Apple

apple.com

Apple's website is a great example of using colors to contrast between the foreground and background. The design is uses different font sizes to emphasize the focus of each element. The fonts colors and background work together to enhase with readability.

Repetition

Canva

canva.com

This website uses proper repetion of element design to help with the navigation of the website. The layout of the website helps to user know the relationship between the title and the coming images/columns. The font repeats the same style and colors to enhanve the user experience.
